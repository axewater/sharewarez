Discord Notifications Help

Setting Up Discord Notifications

Step 1: Create a Discord Webhook

  1. Open your Discord server settings
  2. Navigate to "Integrations"
  3. Click on "Create Webhook"
  4. Name your webhook (e.g., "SharewareZ Notifications")
  5. Choose the channel where notifications should appear
  6. Copy the Webhook URL

Step 2: Configure SharewareZ

  1. Go to Discord Settings in SharewareZ admin panel
  2. Paste your Webhook URL
  3. Set your preferred bot name
  4. Optionally, add a bot avatar URL
  5. Save your settings

Available Notifications

  • New Games: Notification when new games are added to the library
  • Game Updates: Notification when game updates are available
  • Downloads: Notification when downloads are processed and ready

Testing Your Setup

After configuring your webhook, you can test it by:

  1. Adding a new game to your library
  2. Checking your Discord channel for the notification
  3. Verifying that all information is displayed correctly
Note: Make sure your webhook URL is kept private and secure.
