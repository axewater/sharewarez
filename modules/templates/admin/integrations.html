IGDB API Settings

How to Get Your IGDB API Credentials
  1. Go to the Twitch Developer Console
  2. Log in with your Twitch account (or create one if needed)
  3. Click on "Register Your Application"
  4. Fill in the application details:
    • Name: Your choice (e.g., "MySharewareZ")
    • OAuth Redirect URL: http://localhost (if testing locally)
    • Category: Select "Website Integration"
  5. After registering, you'll receive your Client ID
  6. Click "New Secret" to generate your Client Secret
Note: IGDB is owned by Twitch, which is why credentials are obtained through the Twitch Developer Console. The free tier includes 4 requests per second and 500k requests per month.

Discord Bot Settings Notification Settings

Setting Up Discord Notifications
  1. Create a Discord Webhook:
    • Open your Discord server settings
    • Navigate to "Integrations"
    • Click on "Create Webhook"
    • Name your webhook (e.g., "SharewareZ Notifications")
    • Choose the channel where notifications should appear
    • Copy the Webhook URL
  2. Configure SharewareZ:
    • Paste your Webhook URL below
    • Set your preferred bot name
    • Optionally, add a bot avatar URL
    • Save and test your settings
Available Notifications: New games, game updates, downloads, and game extras. Configure which notifications to send in the Notification Settings page.

How to Get SMTP Server Information
Option 1: Use Your Internet Service Provider's SMTP Server (Recommended)

Most Internet Service Providers (ISPs) provide SMTP server access. This is typically the most reliable option for most users.

  • Check your ISP's support website or documentation
  • Contact your ISP's customer support
  • Common format: smtp.yourisp.com
Option 2: Use Gmail SMTP

Gmail provides SMTP access, but requires some additional setup:

  • Server: smtp.gmail.com
  • Port: 587
  • Security: TLS
  • You'll need to enable "Less secure app access" or create an App Password
Option 3: Professional SMTP Services

These services offer reliable email delivery with higher limits:

  • SendGrid
  • Amazon Simple Email Service
  • Mailgun
  • SMTP2GO
Note: Always use TLS/SSL encryption when available, and verify the port numbers with your provider.
