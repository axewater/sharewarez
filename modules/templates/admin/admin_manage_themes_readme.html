Creating Your Own Theme

Theme Structure

A theme consists of the following components:

  • A ZIP file containing all theme files
  • A theme.json file with theme metadata
  • CSS files in a css folder
  • Optional image files in an images folder

theme.json Format


{
    "name": "Your Theme Name",
    "author": "Your Name",
    "description": "A brief description of your theme",
    "version": "1.0.0",
    "release_date": "YYYY-MM-DD"
}

CSS Files

Place your CSS files in the css folder. The main CSS file should be named base.css. Additional CSS files are structured by folder.

Creating the ZIP File

Important: The ZIP file must contain the theme files at the root level, not inside a subfolder.

  1. Create a folder with your theme name for organization
  2. Add the theme.json file to this folder
  3. Create a css subfolder and add your CSS files
  4. Optionally, create an images subfolder for any theme-specific images
  5. Important: When creating the ZIP file, select all the contents of your theme folder (theme.json, css/, images/, etc.) and zip those files directly - do NOT zip the theme folder itself

Correct ZIP Structure

Your ZIP file should contain:


mytheme.zip
├── theme.json          ← At root level
├── css/               ← At root level
│   ├── base.css
│   ├── components.css
│   └── admin/
│       └── admin_dashboard.css
└── js/                ← Optional, at root level
    └── custom.js

Incorrect ZIP Structure (will fail)


mytheme.zip
└── mytheme/           ← Extra folder layer causes upload to fail
    ├── theme.json
    ├── css/
    └── js/

Uploading Your Theme

Use the theme upload form on the Manage Themes page to upload your ZIP file.
