Help / FAQ

Thank you for participating in our closed beta phase. This guide will help you navigate and make the most of our platform.

User Guide

  • Browse our extensive games library using filters or the search function.
  • Simply start typing to initiate a search.
  • Game downloads are queued for processing. Once selected, our server will prepare a ZIP file for you to download.

Administrator Guide

  • Utilize the auto folder scanner to efficiently scan your game directories.
  • The folder scanner will process all subfolders within the specified path, attempting to add each as a game to the library.
  • Ensure each scanning job is completed before initiating another to avoid conflicts.
  • You have the flexibility to add any folder accessible by your server.
  • Game names are automatically sanitized for consistency.
  • After scanning, review the unmatched folder manager for any discrepancies.
