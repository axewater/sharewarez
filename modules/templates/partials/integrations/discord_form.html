How to Get a Discord Webhook URL
Step 1: Open Your Discord Server

Navigate to the Discord server where you want to receive notifications. If you don't have a server, create one first.

Step 2: Access Server Settings
  • Right-click on the server name (or click the dropdown arrow next to it)
  • Select Server Settings from the menu
  • You must have Manage Webhooks permission to create webhooks
Step 3: Navigate to Integrations
  • In the left sidebar of Server Settings, click Integrations
  • Click on Webhooks or Create Webhook
Step 4: Create a New Webhook
  • Click the New Webhook button
  • Choose the channel where you want notifications to appear
  • Optionally customize the webhook name and avatar (these can also be configured in SharewareZ)
Step 5: Copy the Webhook URL
  • Click on your newly created webhook to expand its settings
  • Click the Copy Webhook URL button
  • Paste this URL into the "Discord Webhook URL" field above
Webhook URL Format:
Your webhook URL should look like this:
https://discord.com/api/webhooks/[WEBHOOK_ID]/[WEBHOOK_TOKEN]
Security Note:
Keep your webhook URL private! Anyone with this URL can send messages to your Discord channel.
